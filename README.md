AI-Based Automated Target Detection for AFVs
AI-powered real-time threat detection for Armoured Fighting Vehicles (AFVs).

 Project info :
 A real-time AI system that analyzes tactical images to identify threats like camouflaged personnel, military vehicles, and concealed positions.

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
